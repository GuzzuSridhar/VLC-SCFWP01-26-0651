1. What is HTML?
HTML (HyperText Markup Language) is used to structure content on web pages.

2. What is a tag in HTML?
A tag defines an element. Example: <p>Paragraph</p>.

3. What is the basic structure of an HTML document?
<!DOCTYPE html>
<html>
  <head>
    <title>Page</title>
  </head>
  <body>
    Content
  </body>
</html>

4. What does <!DOCTYPE html> do?
It tells the browser that the document is HTML5.

5. What is the <head> tag used for?
It contains metadata like title, links, and scripts.

6. What is the <body> tag?
It contains visible content on the page.

7. What is a paragraph tag?
<p> defines a paragraph.

8. What are headings in HTML?
<h1> to <h6> define headings (h1 = largest).

9. What is an attribute?
Extra information added to a tag.
Example: <img src="image.jpg">


10. What is the <a> tag?
It creates hyperlinks.
<a href="https://example.com">Visit</a>

11. What is the <img> tag?
Displays images.
<img src="image.jpg" alt="description">

12. What is the purpose of alt in images?
Provides alternative text for accessibility.

13. What is a list in HTML?
Used to group items.

14. Types of lists?
Ordered (<ol>)
Unordered (<ul>)

15. What is <li>?
List item.

16. What is a <div>?
A block container used for layout.


17. What is a <span>?
An inline container.

18. Difference between <div> and <span>?
<div> = block
<span> = inline

19. What is a form?
Used to collect user input.

20. What is <input>?
Form field element.


21. Types of input fields?
text, password, email, checkbox, radio, etc.


22. What is <button>?
Clickable button.

23. What is <br>?
Line break.

24. What is <hr>?
Horizontal line.

25. What is semantic HTML?
Tags with meaning like <header>, <footer>, <article>.

26. What is CSS?
CSS (Cascading Style Sheets) styles HTML elements.

27. How to apply CSS?
Inline
Internal
External

28. Inline CSS example:
<p style="color:red;">Text</p>

29. Internal CSS example:
<style>
  p { color: red; }
</style>

30. External CSS example:
<link rel="stylesheet" href="style.css">

31. What is a selector?
Targets HTML elements.
Example: p {}

32. What is a property?
Defines what to change (e.g., color).

33. What is a value?
Defines how to change (e.g., red).

34. Example of CSS rule:
css
p {
  color: blue;
}

35. What is color property?
Changes text color.

36. What is background-color?
Changes background color.

37. What is font-size?
Controls text size.

38. What is margin?
Space outside an element.

39. What is padding?
Space inside an element.

40. Difference between margin and padding?
Margin = outside
Padding = inside

41. What is border?
Defines outline of element.

42. What is display property?
Controls layout behavior.

43. Common display values?
block
inline
inline-block
none

44. What is flexbox?
Layout system for aligning elements.

45. What is grid?
2D layout system.

46. What is class in CSS?
Reusable selector.
<p class="text"></p>

47. What is id?
Unique identifier.
<p id="main"></p>

48. Difference between class and id?
class = reusable
id = unique

49. What is overflow?
Controls content spilling outside container.

50. What is position property?
Controls element placement:
* static
* relative
* absolute
* fixed
